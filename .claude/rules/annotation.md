# 标注 Annotation

**何时读:** 在 RM 工具里给作文加标注时。

**真源:** 本文件是标注用法的唯一真源。标注是**评分证据**,不是分数本身——分数存在 `S&A` / `C&S` 字段里(见 `paper-structure.md`)。

---

## 标注是证据,不是分数

标注层用来**展示判断背后的依据**,让分数可追溯。它不替代评分决策,也不驱动最终分。安全流程:

```text
读 response → 标出有用的证据 → 决定 S&A 和 C&S → 录分 → 核对总分 → 保存
```

## 左侧标注工具条(RM Assessor 实测)

自上而下,红色那组是标注工具(上面深蓝的是导航 / 缩放 / 全屏,不是标注):

| 图标 | 官方名 | 用途 |
|---|---|---|
| `SEEN` | **SEEN** | 标记此页已读。**每页都打**(放页眉空白处)。 |
| `?` | **?** | 意思不清、表达令人困惑处。 |
| 〜 | **H Wavy**(波浪线) | **画出语法 / 措辞错误**(病句、用词不当、别扭表达)。 |
| 🖊 | **Highlight**(笔) | **标出错别字**(写错的字)。 |
| `^` | **^**(caret) | 遗漏 / 缺失 / 需插入处。 |
| ✓ | **Tick** | 有效的点、清晰的想法、有效的措辞。 |
| `DET` | **DET** | 有用的细节、展开、支撑材料。 |

工具开关:右侧面板顶部 `Annotations: ON`——标注层打开时这些标记才可见 / 可加。

## SEEN 每页都打(housekeeping 纪律)

实测样本(6158269758、6156397880)里 **每一页页眉都有 `SEEN`**,包括题目页、写满的作文页、以及**空白方格页**。

- 默认对**每一页**打 `SEEN`,表示"这页我读过了"。
- 空白 / 学生没写满的页:只打 `SEEN`,不加别的。
- 写了内容的页:打 `SEEN` 之外,再按下面的纪律稀疏加证据标记。

## 其余标记的纪律

- 证据标记**稀疏但有代表性**,不要逐行标。
- **错别字 → `Highlight`(笔)**:把写错的字标出来。
- **语法 / 措辞错误 → `H Wavy`(波浪线)**:把病句、用词不当、别扭表达画出来。
- `Tick` / `DET` 标正向证据(有效点 / 有用展开)。
- `?` 标意思不清 / 困惑处;`^` 标缺失 / 插入 / 影响判断的空缺处。
- 当分数取决于一个 **judgement call** 时,正负标记都放几个,让档位有据可依。
