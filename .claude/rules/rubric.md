# 评分标准 Rubric — S&A /12 + C&S /13

**何时读:** 给任何一篇作文定 band、定分时。

**真源:** 本文件是 band 描述的唯一真源。评分对象是**中文写作**——band 里的「字词 / 语法 / 标点 / 语体(register)」错误指中文文本本身。校准用 `anchors.md`,录分流程用 `paper-structure.md`。

---

## S&A /12 — Style and Accuracy(语言)

### 11–12 — 非常流畅、有控制

- 句子掌控成熟。
- 词汇多样、表达准确。
- 语气 / 语体稳定。
- 错误轻微,不影响阅读。

### 9–10 — 清晰、流畅、基本准确

- 词汇有范围,常常有效。
- 措辞有一定企图心。
- 有错误但很少影响沟通。
- 全文质量稳定。

### 7–8 — 大体清晰,但有明显局限

- 意思总体可懂。
- 有些复杂结构能用,但控制不均。
- 词汇够用而不精确。
- 错误与别扭措辞**经常**出现,但沟通通常仍成立。

### 5–6 — 简单、不均

- 意思有时不清。
- 句式可能重复。
- 词汇有限。
- 字词、语法或标点错误开始影响清晰度。

### 3–4 — 大部分难懂

- 基础词汇。
- 语言错误频繁。
- 语体与语气薄弱。
- 主旨或许仍能还原。

### 1–2 — 非常难懂

- 句子构造差。
- 字词与语法错误持续。
- 几乎无语体 / 语气掌控。

## C&S /13 — Content and Structure(内容与结构)

### 11–13 — 强力完成任务

- 目的与读者对象清晰。
- 想法有展开、可持续、衔接好。
- 结构稳固、连贯。
- 描写 / 叙述类:画面与氛围鲜明且有控制。

### 9–10 — 良好完成任务

- 组织清晰。
- 相关想法有展开。
- 段落 / 阶段大体顺序得当。
- 细节有效,但不总能贯穿到底。

### 7–8 — 充分完成任务

- 明确尝试切题。
- 想法相关,但常较直白。
- 有一些展开,但不深、不一致。
- 顺序总体可接受,偶有弱衔接。

### 5–6 — 对任务有一定投入

- 有相关要点 / 细节,但有限。
- 展开简单或重复。
- 结构可能松散。
- 文章可能开头比结尾强,或反之。

### 3–4 — 投入非常有限

- 只有少数相关要点。
- 几乎无展开。
- 顺序薄弱。
- 即使有分段也帮助不大。

### 1–2 — 投入极少

- 想法难以跟随。
- 几乎无展开。
- 结构混乱。

## 定 band 时的提醒

- 「字数长 ≠ 高分」——见 `anchors.md` 的 calibration rules。
- 语言强 / 内容弱(或反之)是允许的,两个 sub-mark 各自独立判。
- 拿不准两个 band 之间时,调出 `anchors.md` 里相邻分数的 anchor 作对比。
